# 🛠️ STACK TECNOLÓGICO ESTRICTO

1. **Framework Core:** Astro. Construye una arquitectura modular (componentes `.astro`, layouts) preparada para la "Astro Islands Architecture" de cara a un futuro sistema de reservas dinámico.
2. **Estilizado:** Tailwind CSS. Utiliza clases de utilidad para todo el diseño.
3. **Animaciones:** CSS3 puro, transiciones de Tailwind y Vanilla JS ligero (Intersection Observers) para efectos de scroll. Prohibido usar librerías pesadas que bloqueen el hilo principal. Rendimiento estricto a 60fps.
4. **Tipado:** TypeScript (estricto) para cualquier lógica de componentes.

# 🎨 SISTEMA DE DISEÑO (NEON NOIR & GLASSMORPHISM)

1. **Dark Mode Nativo y Obligatorio:** El fondo base de la web debe ser negro azabache (`#000000`) o grises abisales (`bg-zinc-950`). Prohibidos los fondos blancos o claros.
2. **Paleta de Acentos (Luces Neón):** Utiliza colores hiper-saturados para botones, iconos, palabras clave y efectos de brillo (`drop-shadow`).
   - Fucsia Iridiscente
   - Púrpura Profundo
   - Azul Eléctrico
3. **Tipografía:**
   - *Títulos (H1/H2):* Sans-Serif o Display geométrica, gruesa, mayúsculas, con tracking ajustado. Debe gritar modernidad y contundencia.
   - *Cuerpo:* Sans-Serif limpia (ej. Inter o Poppins), altamente legible en pantallas móviles pequeñas.
4. **Composición:** Usa diseño "Mobile-First". Implementa tarjetas estilo *Glassmorphism* (cristal esmerilado con `backdrop-blur` y bordes semitransparentes `border-white/10`).
